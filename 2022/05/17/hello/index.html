#MySQL
##数据库
1.DB：数据库，保存的容器（像一个仓库）
2.DBMS：数据库管理系统，又称数据库软件，用于管理DB中的数据（相当于一个仓库管理员）
3.SQL：用于和DBMS的一个语言，sql可以用于其他的数据库
##数据库的特点
1.先将数据放到表再放到库中
2.表为列组成，在这里叫字段
##MySQL登录和退出
win自带客户端
登录：mysql -h主机名 -p客户端 -u用户名 -p密码
退出：exit或ctrl+c
##数据库常用指令
1.查询所有数据库：
show databases；
2.查询当前数据库：
select database;(）
3.查询当前库所有表：
show tables;
4.查询其它库的表：
show tables from 库名;
5.创建数据库：
CREATE DATABASE [ IF NOT EXISTS ] 数据库名 [ DEFAULT CHARSET 字符集] [COLLATE 排序规则 ];
6.创建表：
create table 表名（
列名 列类型，
列名 列类型，
）；
7.删除数据库：
DROP DATABASE [ IF EXISTS ] 数据库名;
8.使用数据库：
use 库名;
9.查看表结构：
desc 表名；
10.查看服务器版本：
方法一：select version();
方法二：mysql –version或mysql –V
##查询命令
1.起别名
方式一：
select name AS 名字
select last_name AS 姓,first_name AS 名 from employees;
方式二：
select last_name 姓,first_name 名 from employees;(AS可以用空格代替)
注意点：
错误：
SELECT salary AS OUT put FROM employees
正确：
SELECT salary AS “OUT put” FROM employees(不想让它是关键词可以加个双引号或单引号)
2.去重
SELECT DISTINCT department_id FROM employees;(加个distinct)
3.连接
sql里的＋只能用于运算,SQL里连接用concat
SELECT CONCAT （’a’，’b’） AS 别名
